# KnowIThub — Internal Knowledge & Communication Platform

A full-stack web application designed to centralize internal communication, 
document management, and knowledge sharing for teams and departments.

Built with Node.js, Express, and a modern frontend stack.

## 🚀 Live Demo
[knowithubv2.onrender.com]

## 💡 Why I Built This
Teams waste hours searching for information scattered across emails, 
chats, and folders. KnowIThub solves this by giving every team 
a single, organized, searchable hub.

## ✨ Key Features
- 📢 Flash banner system for urgent announcements
- 🔍 Searchable content repository organized by department
- 💬 Integrated comment system for team discussions
- 📁 File sharing and document management
- 📞 Phone directory
- 💻 Code snippet library for technical teams
- 🔖 Personal bookmark system
- 🌙 Dark mode support
- 👥 Role-based access (Admin / Regular User)
